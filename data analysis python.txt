import tkinter as tk
from tkinter import filedialog, messagebox
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

def load_data():
    global data
    file_path = filedialog.askopenfilename(title="Select Data File", filetypes=[("CSV files", "*.csv"), ("Excel files", "*.xlsx")])
    if not file_path:
        return
    
    try:
        if file_path.endswith('.csv'):
            data = pd.read_csv(file_path)
        else:
            data = pd.read_excel(file_path)
        messagebox.showinfo("Success", "Data loaded successfully!")
    except Exception as e:
        messagebox.showerror("Error", f"Could not load file: {e}")

def show_summary():
    if data is None:
        messagebox.showwarning("Warning", "Please load a dataset first.")
        return
    summary = data.describe()
    messagebox.showinfo("Data Summary", summary.to_string())

def check_missing_values():
    if data is None:
        messagebox.showwarning("Warning", "Please load a dataset first.")
        return
    missing = data.isnull().sum()
    messagebox.showinfo("Missing Values", missing.to_string())

def plot_correlation():
    if data is None:
        messagebox.showwarning("Warning", "Please load a dataset first.")
        return
    
    plt.figure(figsize=(8, 6))
    sns.heatmap(data.corr(), annot=True, cmap="coolwarm", fmt=".2f")
    plt.title("Correlation Heatmap")
    plt.show()

data = None

root = tk.Tk()
root.title("Data Analysis Tool")
root.geometry("500x300")

frame = tk.Frame(root)
frame.pack(pady=20)

load_button = tk.Button(frame, text="Load Data", command=load_data)
summary_button = tk.Button(frame, text="Show Summary", command=show_summary)
missing_button = tk.Button(frame, text="Check Missing Values", command=check_missing_values)
correlation_button = tk.Button(frame, text="Plot Correlation", command=plot_correlation)

load_button.grid(row=0, column=0, padx=10)
summary_button.grid(row=0, column=1, padx=10)
missing_button.grid(row=1, column=0, padx=10)
correlation_button.grid(row=1, column=1, padx=10)

root.mainloop()







Q1: Can you explain the purpose of this project?
✅ Answer:
This project is a Data Analysis Tool that allows users to load datasets, perform exploratory data analysis (EDA), and visualize data trends. It helps users analyze missing values, summary statistics, and correlations efficiently using a simple graphical user interface (GUI).

2. Python & Tkinter Questions
Q2: Why did you choose Tkinter for the GUI?
✅ Answer:
I chose Tkinter because:

It is built-in with Python (no extra installation needed).

It is lightweight and sufficient for simple desktop applications.

It allows easy integration with pandas and matplotlib for data visualization.

Q3: How does the filedialog.askopenfilename() function work?
✅ Answer:
The filedialog.askopenfilename() function opens a file selection dialog and lets the user select a file. The selected file’s path is returned, which we then use to load data into a pandas DataFrame.

Q4: How do you update the GUI based on user interactions?
✅ Answer:
I use Tkinter event-driven programming, where each button is linked to a function (command=some_function). When a user clicks a button, the associated function processes the data and updates the GUI using message boxes and plots.

3. Pandas & Data Analysis Questions
Q5: How do you handle missing values in this project?
✅ Answer:
I check for missing values using data.isnull().sum(), which returns the count of missing values in each column. We currently only display this information, but in a real-world scenario, we could:

Fill missing values (data.fillna(value)).

Drop missing rows (data.dropna()).

Use interpolation (data.interpolate()).

Q6: What does data.describe() do?
✅ Answer:
data.describe() generates summary statistics for numerical columns, including:

Count (number of non-null values).

Mean, Std Dev (average and spread of data).

Min, Max (range of values).

25%, 50%, 75% quartiles (distribution).

4. Visualization & Seaborn Questions
Q7: What is a correlation heatmap, and why is it useful?
✅ Answer:
A correlation heatmap visually represents the correlation between numerical variables in a dataset.

Positive correlation (close to 1): Variables increase together.

Negative correlation (close to -1): One variable increases while the other decreases.

Close to 0: No correlation.
This helps in feature selection and pattern identification in data science projects.

Q8: How does seaborn.heatmap(data.corr()) work?
✅ Answer:

data.corr() computes the Pearson correlation between numerical columns.

sns.heatmap() visualizes the correlation matrix as a heatmap.

annot=True displays correlation values on the heatmap.

cmap="coolwarm" sets the color gradient for better visibility.

5. Error Handling & Optimization Questions
Q9: What error handling mechanisms have you implemented?
✅ Answer:
I use try-except blocks when:

Loading files (try-except to handle file errors).

Checking for missing columns (using if 'Column' not in data.columns:).

Displaying warnings (messagebox.showwarning() if no data is loaded).

Q10: How can this project be optimized for handling large datasets?
✅ Answer:

Use Dask instead of Pandas for handling large datasets efficiently.

Load only required columns (usecols=['col1', 'col2'] in read_csv()).

Use .sample() to analyze subsets of data instead of the entire dataset.
