# Data-analysis-Application
Optimized data ingestion workflows, allowing users to effortlessly parse datasets, compute statistical summaries, and identify core data correlations. Designed and executed automated scripts to locate and visualize missing values, mitigating data preprocessing bottlenecks for downstream analysis
# Python Data Analysis Application

A simple data analysis tool built using Python, Pandas, and Matplotlib.

## Features
- Load CSV datasets
- Display dataset information
- Generate statistical summaries
- Handle missing values
- Visualize data distributions
- Export cleaned data

## Installation

pip install -r requirements.txt

## Run

python data_analysis.py

## Technologies Used
- Python
- Pandas
- Matplotlib
